# Network Packet Sniffer using Python

## Internship Task - CodeAlpha

## Objective
To build a Python program that captures live network traffic packets and analyzes
their structure to understand network protocols and data flow.

## Tools & Technologies
- Python 3
- Scapy
- Npcap (Windows)
- Visual Studio Code

## Featuress
- Captures live network packets
- Displays source and destination IP addresses
- Identifies protocols such as TCP, UDP, DNS
- Shows payload information
- Supports IPv4 and IPv6 traffic

## How to Run
1. Install Python and Scapy
2. Install Npcap (for Windows)
3. Run VS Code as Administrator
4. Execute:
5. Generate traffic using a browser or ping command

## Learning Outcomes
- Understanding of packet structure
- Real-time network traffic analysis
- Knowledge of TCP, UDP, DNS, and IP protocols
