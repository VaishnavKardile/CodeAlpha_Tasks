# Task 3: Secure Coding Review

## Internship Task – CodeAlpha

## Objective
To perform a secure coding review of a Python-based application and identify
security vulnerabilities along with recommended remediation steps.

## Application Reviewed
A Python user management system that includes registration, login, user listing,
and deletion functionalities.

## Review Focus Areas
- Authentication security
- SQL injection risks
- Password handling
- Input validation
- Authorization controls
- Secure coding practices

## Files Included
- `vulnerable_app.py` – Application under review (intentionally insecure)
- `Secure_Coding_Review_Report.md` – Detailed security analysis and recommendations

## Outcome
This task improved understanding of common coding vulnerabilities and reinforced
the importance of secure software development practices.
